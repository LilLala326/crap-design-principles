# crap-design-principles
A website dedicated to exploring the C.R.A.P. design principles (Contrast, Repetition, Alignment, and Proximity) and their role in creating effective, user-friendly interfaces. The site also covers web accessibility guidelines (WCAG), ensuring content is designed to be inclusive and usable for all users. Built with a modular PHP structure to maintain consistency across pages. This project was originally developed in collaboration with:

- Anahy Ramierz
